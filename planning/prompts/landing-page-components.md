# Landing Page Component Prompts

## 1. Hero Section
```
Create a responsive hero section component for AlfaRetailers.com:

Requirements:
- Full-width background image overlay
- Headline: "Your Property's Been Sitting Empty? We'll Turn It Into a 24/7 Income Generator"
- Subheadline: "From stagnant long-term rental to profitable Airbnb in under 30 days"
- Primary CTA: "Get Your Free Rental Analysis" (orange button #ea580c)
- Trust badges below: "50+ Properties Managed", "$2M+ in Owner Earnings", "Average 40% Income Increase"

Make it:
- Mobile-first responsive
- With smooth scroll animations using Framer Motion
- Using the color scheme: primary blue #2563eb, accent orange #ea580c
- Include loading states and accessibility features
```

## 2. How It Works Section
```
Implement the How It Works section with 3-step process visualization:

Steps:
1. Apply: Submit your property for free evaluation
2. Evaluate: We analyze your property's potential and create strategy
3. Launch: We list and manage your property on Airbnb

Requirements:
- Visual icons or illustrations for each step
- Brief descriptions
- Mobile-first responsive grid layout
- Subtle hover animations
```

## 3. Income Calculator
```
Create an interactive income calculator component that estimates potential earnings:

Inputs:
- Property type (dropdown)
- Number of bedrooms
- City/location
- Amenities checkboxes (pool, wifi, parking, etc.)

Output:
- Estimated monthly income from short-term rental
- Comparison with long-term rental
- ROI percentage

Requirements:
- Real-time updates as user changes inputs
- Professional styling matching the site theme
- Mobile-friendly interface
- Add "Get Detailed Analysis" CTA
```
